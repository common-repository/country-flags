Insert country flags via a shortcode. Example: [flag country="fr"] inserts corresponding image size.


== Description ==

Flag icons as served by [countryflags.io](https://countryflags.io) API by warrigal.studio.

Usable icon size is one of 16, 24, 32, 48, and 64. Default icon size is 16.
253 contry level domains (cld) supported to identify a country.

= Usage =

 * [flag country="fr"] 
 * [flag country="fr" size="16"] 


== Installation ==

Search and activate the plugin named: "country flags with countryflags.io".


== Frequently Asked Questions ==

= What is the source of flag images? =
* See [countryflags.io](https://countryflags.io) for details.

= Does it download country flags images for offline use? =
No. Rather it links to the the images to their webiste on realtime.

= What are the valid sizes?
Use size="XX" where XX is one of 16, 24, 32, 48, 64. Units in pixels.
